
# Barangay Mapatag Senior Profiling System

A high-performance digital platform for managing senior citizen records, medical alerts, and benefit distributions.

## Deployment to Vercel
1. Push this code to a GitHub repository.
2. Go to [Vercel](https://vercel.com) and click **"Add New" > "Project"**.
3. Import your repository.
4. **Environment Variables**: Add `API_KEY` with your Google Gemini API Key.
5. Click **Deploy**.

## Local Development
1. Install dependencies: `npm install`
2. Run development server: `npm run dev`
3. Build for production: `npm run build`

## Features
- **Registry**: Comprehensive search and filtering of senior citizens.
- **Medical Alerts**: Real-time tracking of high-risk health conditions.
- **AI Insights**: Automated health summaries via Google Gemini.
- **Audit Logs**: Full transparency of system modifications.
- **Responsive UI**: Optimized for tablets and desktop administrative use.
